## Installation

Clone the project and go inside

Run:

```bash
  npm install
```

To launch the project, use

```bash
  npm start
```

    
To compile less style, you can use less or less-watch-compiler

```bash
  lessc styles.less styles.css
```
```bash
  less-watch-compiler [options] <source_dir> <destination_dir>
```
